## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/smart-market.git
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. For production build:
```bash
npm run build
npm run preview
```
